## Code Snippets

Inline code example:
`const lorem = "ipsum dolor sit amet";`

### Code Block (JavaScript)

```javascript
function loremIpsum(count) {
  let text = "";
  for (let i = 0; i < count; i++) {
    text += "Lorem ipsum ";
  }
  return text.trim();
}

console.log(loremIpsum(5));
```

### Code Block (Python)

```python
def lorem_ipsum(n):
    return " ".join(["lorem"] * n)

print(lorem_ipsum(5))
```
